    Why

    Public orbital data deserves durable, independently verifiable memory outside any single service.

    What

    A daily archive of tracked artificial objects in Earth orbit, made visible as a living record.

    How

    Independent nodes reproduce each snapshot, compare SHA-256 fingerprints, and publish permanent evidence.
